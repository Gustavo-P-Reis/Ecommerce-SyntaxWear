# SyntaxWear - Tênis e Sneakers Online

## Descrição do Projeto
Este projeto é um protótipo de e-commerce focado na venda de tênis e sneakers, desenvolvido com HTML5 e CSS3. O objetivo é apresentar uma interface moderna e responsiva, com seções dedicadas a categorias de produtos, destaques e informações da loja.

## Funcionalidades
- **Header Fixo e Responsivo:** Navegação principal com logo, links de categorias (Masculino, Feminino, Outlet) e acesso rápido (Minha conta, Ajuda, Carrinho). Adaptável para diferentes tamanhos de tela com um menu hambúrguer para dispositivos móveis.
- **Hero Section Dinâmica:** Uma seção de destaque com imagem de fundo, título impactante e botões de chamada para ação ("Ver modelos", "Comprar").
- **Categorias de Produtos:** Exibição de categorias de tênis (Casual, Esporte, Moderno, Futurista) com cards interativos.
- **Product Grid (Grade de Produtos):** Layout de grade com diferentes produtos em destaque, incluindo informações como título, subtítulo e botões de ação.
- **Footer Completo:** Contém formulário de inscrição para newsletter, links para redes sociais e navegação secundária organizada por categorias (Masculino, Feminino, Outlet, Nossas Lojas, Sobre).

## Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização completa, incluindo:
    - Reset CSS (baseado no reset do Andy Bell) para padronização de estilos.
    - Variáveis CSS para gerenciamento de cores e fontes.
    - Estilos base para elementos globais.
    - Componentes CSS modulares para Header, Hero, Categorias, Grade de Produtos e Footer.
    - Design responsivo utilizando Media Queries para adaptação a diferentes dispositivos.

## Como Rodar o Projeto
Para visualizar o projeto, basta abrir o arquivo `index.html` em seu navegador web. Não é necessário nenhum servidor local.

## Responsividade
O layout do site é totalmente responsivo, adaptando-se a diferentes tamanhos de tela, desde desktops até dispositivos móveis, garantindo uma ótima experiência de usuário em qualquer aparelho.
